Report card comment generator

How to use the report card comment generator

To use the generator, you need to input a couple of things:

  1. File containing student names, genders, overall grade, and 1 learning skill to improve on
  2. Course name (eg. "Grade 8 Biology", "Media Studies", "MCV4U", etc.)
  3. Name of 2 assignments to highlight in the comments
  4. Learning outcome of 2 assignments to highlight
  5. Whether this is the final report card (this adds a well-wishing statement)

Gradebook template

You can download this template on the main page.

Additional notes:

  • Please ensure your file is saved in the csv format. Csv or Comma-separated values is a computer-friendly format that allows this app to read your spreadsheet data.
  • Student names should be typed as you would like it to be written in their comments
  • Gender is not case sensitive, but if you put anything other than "f" or "m" the pronouns will default to they/theirs
  • Overall mark must be a number from 0 to 100
  • "Area to improve" can be one of the 6 learning skills. Please check that your inputs are spelled correctly
  • Once you click "Submit", your comments will be generated below and you can copy + paste them from this page
  • There is currently no functionality for the assignment marks so you can put whatever you want there. Hopefully I'll get around to adding this.
